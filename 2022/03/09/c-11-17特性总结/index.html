c++11/17特性总结

C++ 11新特性

右值引用

Lambda表达式

参考：dsw846

C++11的一大亮点就是引入了Lambda表达式。利用Lambda表达式，可以方便的定义和创建匿名函数，简化STL用到的函数对象或指针。

格式

Lambda表达式完整的声明格式如下

[capture list] (params list) mutable exception-> return type { function body}

其中可以省略部分成分，最常用的有：[capture list] (params list) -> return type {function body}，另外也有如下两种形式：

  • [capture list] (params list) {function body}，自动判断返回类型
  • [capture list] {function body}，类似于无参函数。

捕获形式(capture list)

  • 值捕获

    值捕获不能对其进行修改，否则编译器会报错。如果在函数内部想要进行修改捕获的值变量，需要加上mutable关键词。

  • 引用捕获

  • 隐式捕获

    隐式捕获，指让编译器根据函数体中的代码自行推断需要捕获的变量的方式

    隐式捕获有两种方式，分别是[=]和[&]。[=]表示以值捕获的方式捕获外部变量，[&]表示以引用捕获的方式捕获外部变量。

  • 混合捕获

    [] // 沒有定义任何变量。使用未定义变量会引发错误。
    [x, &y] // x以传值方式传入（默认），y以引用方式传入。
    [&] // 任何被使用到的外部变量都隐式地以引用方式加以引用。
    [=] // 任何被使用到的外部变量都隐式地以传值方式加以引用。
    [&, x] // x显式地以传值方式加以引用。其余变量以引用方式加以引用。
    [=, &z] // z显式地以引用方式加以引用。其余变量以传值方式加以引用。

参数注意事项

Lambda表达式中传递参数有一些限制：

  • 参数列表中不能有默认参数
  • 不支持可变参数
  • 所有参数必须有参数名
